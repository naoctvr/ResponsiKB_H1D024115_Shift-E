# Responsi Kecerdasan Buatan (KB)

Website ini adalah implementasi sistem cerdas menggunakan **Logika Fuzzy** dan **Sistem Pakar** untuk memenuhi tugas Responsi Kecerdasan Buatan.

## 👤 Identitas Mahasiswa
- **NIM:** H1D024115
- **Angkatan:** 2024

## 🚀 Fitur Utama
1.  **Fuzzy Diet Evaluator**: Menentukan kelayakan menu diet berdasarkan kandungan Kalori, Protein, Lemak, Gula, dan Serat menggunakan logika fuzzy sederhana.
2.  **Skin Care Expert System**: Diagnosa masalah kulit wajah dan rekomendasi perawatan menggunakan pendekatan *Rule-Based System*.
3.  **Heuristic Search Integration**: Fitur pencarian makanan yang secara otomatis memetakan kata kunci ke nilai nutrisi awal.

## 🛠️ Teknologi
- **HTML5**: Struktur semantik.
- **Vanilla CSS**: Desain Glassmorphism & Responsive.
- **Vanilla JavaScript**: Logika sistem pakar dan fuzzy tanpa framework eksternal.
- **Font Awesome**: Ikonografi.

## 📁 Struktur File
- `index.html`: Struktur utama website.
- `style.css`: Desain, animasi, dan layouting.
- `script.js`: Logika cerdas (Fuzzy & Expert System).
